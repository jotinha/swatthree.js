This is an experimental WebGL map viewer for SWAT3 scn files. Built on top of three.js and one or two sleepless nights.

The scn file spec was never released, and I'd be very surprised if it ever was, but with a bit of patience and free time it's been almost fully reverse engineered.

See on github.
